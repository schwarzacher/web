Numerical Part

Part of this project is a numerical computations. The computations will verify the theoretical results. The aim is to compute rebound of an ellastic structure in a fluid. The code is based on finite element method, which is based on weak formulation of the problem. The discretization in finite element method is based on a finite mesh describing the domain. We will use Arbitrary Lagrangian-Eulerian (ALE) mothod for dealing with fluid-structure problem.
